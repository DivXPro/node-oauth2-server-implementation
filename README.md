# Node OAuth2 Server Implementation

Supports `    "oauth2-server": "^3.0.0-b2",`

## Installation

```
git clone https://github.com/divxpro/node-oauth2-server-divx
npm install
npm start or node ./bin/www
```

## API

### Grant [/oauth/token]

#### Password Grant
    POST /oauth/token

#### Authorization Grant
    POST /oauth/token

#### Me
    GET /me
